Chenyu Zhao · Ph.D. Student · Nankai University

Biography

I am a Ph.D. student in the College of Software at Nankai University. Before starting my doctoral study, I worked as an R&D engineer at Alibaba Cloud Intelligence Group.

My research interests include AIOps and AgentOps, with a focus on building intelligent systems that can diagnose, recover from, and learn through software failures.

My recent work explores reliable software engineering agents, system-level package repair, language-model-based system building, and failure diagnosis for large-scale software systems.

Research

Reliability and autonomy for modern software systems.

🎯

Research Vision

My research aims to improve the reliability and autonomy of modern software systems by combining software engineering, system operations, and artificial intelligence.

🤖

Reliable Software Agents

I study how software engineering agents fail, how they can preserve useful evidence during execution, and how structured recovery mechanisms can help them complete complex real-world tasks.

⚙️

Operational Intelligence

I work on failure detection, diagnosis, and repair for microservice and cloud systems, with an emphasis on practical methods that can operate under noisy and incomplete production signals.

🧩

System Building with LLMs

I investigate whether language models can go beyond code generation to build, debug, and repair complete software systems.
