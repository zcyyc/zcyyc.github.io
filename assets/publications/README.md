# Publication figures

Place each framework figure in this directory using the filename below. JPG, PNG,
or WebP all work; if you change the extension, update the corresponding `image`
value in `publications.js`.

Recommended image ratio: **16:10** (for example, 1600 × 1000 px).

To add or remove a Paper or Code button, edit that publication's `paper` or
`code` value in `publications.js`. Leave the value empty (`""`) to keep the
optional placeholder button visible.
